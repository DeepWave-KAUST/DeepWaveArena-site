DeepWave Arena - Seismic Processing v0

This benchmark evaluates four tasks on a test set:

  • Denoising: predict syn_high
  • Direct arrival removal: predict syn_high_nodirect
  • Demultiple: predict syn_demult_nodirect
  • Vrms prediction: predict syn_vrms

Participants submit predictions only (no model/code submission).

Data

The evaluator uses a hidden reference file test.npz that contains ground truth for the three targets.

A small sample_public.npz is provided to help you match the required submission structure.

Evaluation

Metrics are computed for each task and averaged across tasks:

  • RMSE
  • MAE
  • SSIM (computed with pytorch_msssim, data range assumed normalized to [-1, 1])
  • PSNR (organizer-defined)

Leaderboard sorting uses: score = -overall_rmse (higher is better).

Submission format

Your submission ZIP must contain a single file:

submission.npz

submission.npz must contain these keys:

  • index (must match the reference)
  • syn_high
  • syn_high_nodirect
  • syn_demult_nodirect

Each prediction array must match the corresponding reference array shape exactly.
